# VALORANT Match Analysis

## What is valorant-match-analysis?
A scripted, written, and presented overview of key metrics in an individual's ranked matches

## Getting Started

### Prerequisites
- [R Studio](https://posit.co/download/rstudio-desktop/)
- [PowerPoint](https://www.microsoft.com/en-us/microsoft-365/powerpoint)
  
### Installation
1. Download the `.ZIP` file and extract the contents: `.CSV`, `.R` script
   - <ins>These two files must be in the same directory level</ins>
2. The contents of the `.R` script are separated by `#`
   - Run the commands below the heading to download any missing packages
   - Freely explore the project to see the visuals presented in the reports
